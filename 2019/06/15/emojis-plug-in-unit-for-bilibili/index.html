bilibili表情插件

Coda · 2019-6-15 · 1000000 次阅读

前言

一直很喜欢bilibili里的表情，于是参考hojun的emoji自己做了一个。
github地址:hexo-tag-emojis-bycoda

demo

使用方式(以本站Sakura主题为例)

安装

有两种方式：

1.直接用npm安装
npm i hexo-tag-emojis-bycoda --save
2.直接去我的github仓库里下载，再在node_modules里新建一个名为hexo-tag-emojis-bycoda文件夹，将解压后的文件夹复制粘贴放入其中即可。

使用

引入CSS

使用之前先要引入css:(建议在文章模板页引入，因为你只在写文章的时候使用表情，按需加载)。当然也可以不引人css，自己写一个css样式，以下的css只是.emoji-coda的表情图片的样式，主要就是控制大小、行内块元素。有更好的css样式可以自由发挥。而且欢迎pr到github仓库。
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/TomoriCoda/hexo-tag-emojis-bycoda/css/emoji.css">
目前有4套表情，但可以根据自己的需要DIY，自定义表情只要向emojis里添加新的文件夹，push再release即可。

注意！每次更新之后需更新cdn的版本：
在node_modules中找到hexo-tag-emojis-bycoda中的index.js 修改版本号即可
如图：

附上我目前的CSS代码:

.emoji-coda {
    display: inline-block !important;
    position: relative;
    width: 56px;
    top: 2px;
    margin: 0 3px !important;
    padding: 0;
}

在文章中使用

语法格式：以 小电视_点赞 为例

{% emoji_coda xiaodianshi/zan %}
参照下面这行代码自行取用呀~
{% emoji_coda 表情包文件夹/表情名 %}
