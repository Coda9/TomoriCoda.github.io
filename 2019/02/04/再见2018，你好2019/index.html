再见2018，你好2019

Coda · 2019-2-4 · 次阅读

2018确实是不太一样的一年。

首先， 18岁了

经历了高考，从高中到了大学
高考确实能给人带来一些独特的感受，经历过，才有所收获。
于我而言，感受到的并不是什么“人间炼狱”，没有什么熬夜备战，没有什么精神紧绷，而是在一定的压力下，真正感受到了自己内心想变得更强的欲望。
那种自己想考入好大学的迫切，与那种“其实高考也不是什么大事”的心理交织，也真的是一种不错的体验呢。
也许回首往事总是觉得风轻云淡吧，现在回看高考：其实也没啥呀。

说完了高考，说说动漫吧，今年上半年看动漫真的是走火入魔了，各种抓紧一切时间看动漫，从每天下课就追紫罗兰国家队药王ggo超能力少女等等新番，到第二天上课作业还没动仍然熬夜补春物夏洛特ab龙女仆小埋（此处省略n部番）等等旧番。
动漫给我确实带来了很大的影响，有感动，有对生命与爱意义的思考，还有那种世界毁灭又和我有什么关系呢？的中二佛系思想23333 还有学园都市差点让我想学脑科学开发大脑？
动漫中的各种让人更加多愁善感？这间接提高我的语文成绩？？
不管怎样，我感受到了 动漫的世界和现实的世界都很美好

今年心态发生了一些转变，看到了特立独行的大佬，觉得有时候不应该追随大多数；一本《当呼吸化为空气》，一本《人间失格》 都带来了独特的触动。

暑假出去走了走，不和家人一起出去其实也不错。在水塘里摸鱼，emmmm第一次体验游泳？

大学从苦逼的军训熬了过来，国庆回去，才发现从此回家的机会真的少了。

大学生活，其实也没多大感受吧，和高中差不了多少，不同的是天天沉迷手机，毕竟各种信息还有和老同学联络（现在基本能克服了哈哈哈），体验了自己工作，发现生活真的不容易。想念以前的老同学，尤其是那个高三一起欢乐的“小团体”？

今年确实认识了比较多的人，发现原来还有人和自己有着那么多的共同爱好 @阿离23333
还有高中时认为lzy应该是那种高冷学霸，都没啥交集，后来发现也是四斋？（不是我带坏的）
总是一起学习的hk大佬

最后，也是最深刻的，知道了很多以前的事，发现各种经历的影响真的很大，造就了现在这种性格的自己。
发现有一个记忆片段一直在脑中，原来一直认为是自己经历过的，今年才知道那不是我自己的经历。hhh真的很诡异，可是这又有什么关系呢?

发现过去18年真的是多灾多难，其实有时候也回想：会不会灾难即将降临？但其实我还是很幸运的，哈哈哈。

不得不提的哔哩哔哩和网易云，真的带来了很多欢笑与感动。

2018 再见，即将到来的2019，初次见面，请多关照。

Q.E.D.
